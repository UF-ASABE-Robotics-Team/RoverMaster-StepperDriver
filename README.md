# RoverMaster StepperController Firmware

### Useful Commands

+ **Build project**

    ```sh
    pio run
    ```

+ **Generate compile_commands.json** (for clangd)

    ```sh
    pio run --target compiledb
    ```

+ **Compile and Upload**

    ```sh
    pio run --target upload
    ```

+ **Clean build files**

    ```sh
    pio run --target clean
    ```
